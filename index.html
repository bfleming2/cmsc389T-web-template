Phone Ownership Management Web-App

Built a full-stack web application using Next.js, React, Javascript, SQL, and Mantine UI library to manage and organize phone ownership across 70 contact centers. Leveraged The Walt Disney Company's ServiceNow REST API to create a C# application that monitored ODBC connections and opened incident tickets if connections were idle for too long. Enhanced .NET Core applications for contact center agents to allow contact center leaders to accurately and promptly close contact center support phone lines. Developed a Jenkins pipeline to create seamless collaboration and iterative development when using Docker and Git.

MakerSpace Manager

Upgraded a Flask and SQLAlchemy web application to allow trained users to access 2500 3D printers and machinery in labs across campus. Developed a system of relational databases of machines, users, logins and user/lab permissions to efficiently store data. Integrated APScheduler package to create background tasks that automatically modify a database to log out users. Utilized UMD's API to access user credentials in order to check permissions and streamline the user experience.

Claro Mentor ETL Process

Created an ETL process in Python for a digital mentorship start-up called Claro Mentor to improve data analysis and data reporting within their business growth division. Designed a process that cleaned and transformed 21 reports from MongoDB to SQL relational databases using MySQL to reduce the run-time of report generation and easily query reports.
